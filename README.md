# fastrelay
Making it easier to run your own Tor Relay Node.

##Usage
Log into server and download the raw fastrely.sh script and then execute it.
`wget https://raw.githubusercontent.com/torworld/fastrelay/master/fastrelay.sh`
`bash fastrelay.sh`
